启发式算法简介

Posted on 2021-01-23

启发式算法简介

基本概念

  • 一种基于直观或经验构造的算法。
  • 利用类似仿生学的原理，将自然中的一些现象抽象成为算法处理相应问题。
  • 不是数学方法，思想更类似于人类解决问题的思想。

常见的启发式算法

例如，我们会经常遇到这种问题：max f(x) x∈X 在某个给定的定义域 X 内，求函数 f(x)对应的最大值。

爬山算法

如图：

爬山算法是一种简单的贪心搜索算法，该算法每次从当前解的临近解空间中选择一个最优解作为当前解，直到达到一个局部最优解。

爬山法是完完全全的贪心法，每次都选择一个当前最优解，因此只能搜索到局部的最优值。

模拟退火算法

概念

退火是一个物理术语，来源于晶体冷却的过程。

模拟退火算法以一定的概率来接受一个比当前解要差的解，因此有可能会跳出这个局部的最优解，达到全局的最优解。

如图，模拟退火算法在搜索到局部最优解 A 后，会以一定的概率接受到 B 的移动。也许经过几次这样的不是局部最优的移动后会到达 C 点，于是就跳出了局部最大值 A。

算法描述

模拟退火算法描述：若 f(i+1) >= f(i) (即移动后得到更优解)，则总是接受该移动；

若 f(i+1)< f(i) (即移动后的解比当前解要差)，则以一定的概率接受移动。“一定的概率”参考了金属冶炼的退火过程，这也是模拟退火算法名称的由来。

根据热力学的原理，在温度为 T 时，出现能量差为 dE 的降温的概率为 P(dE)。P(dE) = exp( dE/(kT) )

遗传算法

基于达尔文的进化论，模拟了自然选择，物竞天择、适者生存。通过 N 代的遗传、变异、交叉、复制，进化出问题的最优解。

在遗传算法中，我们首先需要将要解决的问题映射成一个数学问题。那么这个问题的一个可行解即被称为一条“染色体”。一个可行解一般由多个元素构成，那么这每一个元素就被称为染色体上的一个“基因”。
例：对于函数 x+y+z<100 而言，[1,2,3]、[1,3,2]、[3,2,1]均是这个函数的可行解，即为染色体。这些可行解一共有三个元素构成，那么在遗传算法中，每个元素就被称为一个基因。

相关概念

适应度函数
  • 作用：衡量染色体（可行解）的优劣
  • 适应度函数会给本次迭代中生成的所有染色体打个分，来评判这些染色体的适应度。
  • 然后将适应度较低的染色体淘汰掉，只保留适应度较高的染色体。经过若干次迭代后染色体的质量将越来越优良。
选择/交叉
  • 作用：产生新的染色体
  • 从上一代的染色体中寻找两条染色体。然后将这两条染色体的某一个位置切断，并拼接在一起，从而生成一条新的染色体。
  • 染色体 i 被选择的概率 = 染色体 i 的适应度 / 所有染色体的适应度之和。适应度越高的个体被选择的概率越大。
变异
  • 作用：产生新的基因
  • 交叉能保证每次进化留下优良的基因，但它仅仅是对原有的结果集进行选择。基因还是那么几个，只不过交换了他们的组合顺序。为了解决这一个问题，我们需要引入变异。
  • 当我们通过交叉生成了一条新的染色体后，需要在新染色体上随机选择若干个基因，然后随机修改基因的值。
    从而给现有的染色体引入了新的基因，突破了当前搜索的限制，更有利于算法寻找到全局最优解。

进化过程

  1. 在算法初始阶段，随机生成一组（N 个）可行解，也就是第一代染色体。

  2. 然后采用适应度函数分别计算每一条染色体的适应程度，并根据适应程度计算每一条染色体在下一次进化中被选中的概率。

  3. 通过“交叉”，生成 N 条染色体；

  4. 再对交叉后生成的 N 条染色体进行“变异”操作；

  5. 分别计算 N 条染色体的适应度和下次被选中的概率

    …

总结

为了找出地球上最高的山，一群兔子们开始想办法…

  • 兔子朝着比现在高的地方跳去，找到了不远处的最高山峰。但是这座山不一定是珠穆朗玛峰。这就是爬山法。
  • 兔子喝醉了，随机地跳了很长时间。这期间，它可能走向高处,也可能踏入平地。但是,他渐渐清醒了并朝他踏过的更高方向跳去。这就是模拟退火。
  • 把一群兔子随机放到了地球上的某些地方。他们不知道自己的使命是什么。但是,如果你过几年就杀死一部分海拔低的兔子， 多产的兔子们自己就会找到珠穆朗玛峰。这就是遗传算法。
